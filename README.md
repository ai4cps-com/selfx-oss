#### *SelfX is a Python framework based on Dash for building ML & data science apps in the domain of CPSs*.

Built on top of [Dash](https://dash.plotly.com/), SelfX allows simple implementation of AI tools for CPSs.
Read [our tutorial](https://selfx.ai4cps.com/getting-started). 

- [dash.gallery](https://dash.gallery): Dash app gallery with Python & R code
